# 📌 About

This repository contains **multiple mini frontend projects** created to practice and improve JavaScript skills through hands-on development.

By building these projects you will learn:

- DOM Manipulation  
- Event Handling  
- JavaScript Logic  
- UI/UX Development  
- Responsive Web Design  

These projects are perfect for:

- Beginners learning JavaScript
- Students building portfolios
- Developers practicing frontend skills
